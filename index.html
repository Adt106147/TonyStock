複利

「複利」與單利的區別在於，透過把每段期間獲得的利息併入本金裡，繼續累積利息，來達到利滾利、錢滾錢的效果，複利投資搭配定時定額投入，也是許多人在投資理財的時候都會採用的方式。

有投資 vs 無投資

以台灣平均薪資、退休年齡、平均支出，差異性是否投資。
假設投資年利率以5%作為計算
因為複利的關係，有投資的錢會越滾越多，相反則會持續消耗殆盡。

首先就來談談何謂股票？
股票是一種有價證券，是股份有限公司為籌集資金發給投資者做為公司資本部分所有權的憑 證 ，成為股東以此獲得股息，並分享公司成長或交易市場波動帶來的利潤；但也要共同承擔公司運作錯誤所帶來的風險。
假設今天A、B、C、D、E 5個人合開一家「kkk」有限公司，每人出200萬元，所以公司資本額共1,000萬元，於是這5個人就是股東，分別擁有這家公司的20%股權。假設股票發行時每股面額10元，所以「kkk」股份有限公司一共有100萬股(資本額1,000萬/每股10元)，而且一張股票就是1,000股，所以公司總共有1,000張股票(100萬股/每張股票1,000股)。於是這5個人就每一人擁有200張股票，換句話「kkk」股份有限公司的所有權被分成了1,000份，每個人各擁有200份，以後公司的資產賺錢或賠錢，甚至公司關閉後的清算，都是按照這個比例去分配，所以股票也就是擁有一家公司所有權的證明。擁有股票的人叫做股東，所以股東就是一家公司的擁有者。

x

股票是什麼?

股票簡介

在接觸理財的基礎概念以後，最重要的就是認識「股票」，它的重要性不只是投資，也是現今社會經濟最基礎的運作方式之一。

簡單來說，擁有股票就是擁有那間公司的部分所有權。並且有福同享，有難同當。賺錢時可以依照持股比例分配盈餘。也要承擔虧損風險。

基本面分析:
（又稱基本分析）是評估資產內在價值以及分析將來可能影響價格的因素的一種方法。這種形式的分析以外部事件及影響、財務報表和行業趨勢為基礎。
核心概念:
1. 根據公司競爭力、總經、產業分析
2. 評估公司的獲利能力、成長性、經營風險
3. 了解公司的內含價值與股價的差距，來決定是否買賣內含價值，就是經過你評估後，認定的企業合理價值。
注意事項:
這是假設在公司財報數據並未作假的前提，並且企業價值能實際反映在股價上。
優勢:
1. 背後邏輯的依據很強，讓你有信心長期持有
2. 不容易買貴。
缺點:
1. 即時性低，財報數據一季才公布一次，營收一個月公布一次。
2. 財報可能造假。
3. 企業價值不一定會反映在股價上。

x

技術面分析:
是指研究過去市場上的資訊，預測未來價格的趨勢以及決定投資的策略，也就是說技術分析主要是研究市場過去的資料來預測未來的走向。
這些過去的資料，就是價量走勢的圖表也就是技術分析的根本。
核心概念:
1. 根據公司股價、成交量、價格圖形去分析
2. 評估股價在過去歷史上的統計結果進行交易技術分析的背後，其實是 “統計學” 的結果
注意事項:
1. 技術面分析的重要假設就是歷史會重演並能反映在股價上。
優點:
1. 反應速度快，可以第一時間進場和脫逃。
缺點:
1. 證據性薄弱，記得要適時停損。
2. 反應雖然快，但容易過度反應。

x

籌碼面
就是在市場流通中的股票，觀察股票是誰在背後進行買賣交易，投資人可以從每日的交易成交資料中，分辨出是誰持有股票或者是融資、融券的比例，透過觀察大戶舉例有
1. 三大法人: 外資，自營商，投信(俗稱基金公司)
2. 千張持有人: 直觀來說就是持有某檔股票千張以上投資人
3. 內部關鍵人
投信買賣股票的數量，分析漲跌趨勢及買賣時機。
核心概念:
1. 根據公司法人持股、股東持股、散戶持股、買賣狀況去分析。
2. 跟著少數大戶、大股東操作，能提高獲利的機率。注意事項:
籌碼面分析的重要假設就是市場中比較有錢、持股較多的人，資訊比一般投資人更發達，跟著他們操作容易獲利。
優點:
背後邏輯依據強，資料的準確性高。
缺點:
1. 大戶也有搞錯得時候。
2. 操作承受的虧損幅度通常較大。較難持續跟進。
3. 操作永遠會慢大戶一步，只能撈點湯喝，並且大戶永遠可以一秒間完全改變方向和看法。

x

股票三分析

剛接觸投資，你一定常常聽到的三個單詞。分別代表了各自不同意義的分析方式，也是最主流的三大派。但不要聽到三種方式，就誤會只能選擇一種，投資並不是一道單選題，而是許許多多複合條件構成的環境。其實不只三大分類，還有像是消息面之類的分析方式。我們就先簡單介紹一下這三種吧。

營業利益率 (operating margin) :
營業利益率反應一家公司在本業獲利的能力，有些財報分析會使用毛利率這個項目，差別在於毛利率的計算並未納入營業費用的考量，營業費用包含行銷、管理、研發等非製造成本。營業利益率能更忠實反應出企業的經營績效。

股東權益報酬率(ROE) :
股東權益報酬率(ROE)綜合經營績效、獲利能力及財務槓桿的使用，衡量股東出資的每一塊錢能夠帶來多少的收益，其計算方式為稅後淨利/股東權益，稅後淨利為扣除利息及公司稅賦以後的淨利，為公司最後的盈餘成果。

x

董監持股比例 :

董監持股變化多少能夠反應內部人對公司的看法。
春江水暖鴨先知，公司的前景如何，董監事一定最先知道，如果董監事長期對該公司的持股都維持在很高的水準，那就代表董監事把自己和公司綁在一起，對公司未來的發展十分有信心，就算景氣一時滑落，也會趁機調整體質。如果董監事長期對公司的持股都不高，那代表董監事對公司長期發展並不是十分有信心，甚至有更惡劣的，則是把公司當作撈錢的工具，錢都往自己口袋放，公司不賺錢，持股比例當然不會太高。

成交量 :
有人說 : 「新手看價，老手看量，高手看籌碼」，所謂的量是價的先行指標。
成交量不只顯示市場對於該股票的熱度，同時也能衡量股票的流動性，成交量的多寡經常是法人或大戶決定是否進出某股票的重要考量。

x

流動比率 :
流動資產/流動負債，作為短期償債能力判斷的依歸。然而若僅依流動比率作為判斷依據，當流動比率過大時可能是公司有過多的現金或是存貨，尚未做到最適的營運資金管理及配置。

速動比率 :
(流動資產-存貨-預付費用)/流動負債，流動資產中的存貨與預付費用的流動性較差，在速動比率的分析中予以扣除，針對其中流動性較佳的資產進行分析。

(註 : 流動性指在不損失價值的前提下變現的能力。)

x

本益比 :
股價/每股獲利，可理解為為了獲得多一塊錢的獲利需要付出的代價，本益比的適當與否通常會與同業體質相似的公司進行比較。

帳市值比 :
帳面價值/股價，作為判斷價值股或成長股的依據，根據Fama-French三因子模型，針對美國股市歷史回報率進行實證研究，試圖找出能夠解釋股票市場的平均回報率受到哪些風險溢價因素的影響，其中一個重要因子為帳市值比效應。
